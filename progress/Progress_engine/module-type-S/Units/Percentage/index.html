Module Units.Percentage

Prints a proportion as a percentage (e.g. 42%).

0.      ↦  "  0%"
0.42    ↦  " 42%"
0.9999  ↦  " 99%"
1.      ↦  "100%"

Note: values will be clamped into the range [0., 1.].

Module Units.Bytes

Prints a numeric value as as a byte count.

0                ↦  "   0.0 B  "
999              ↦  " 999.0 B  "
1024             ↦  "   1.0 KiB"
1024 * 1023      ↦  "1023.0 KiB"
1024 * 1024 - 1  ↦  "1023.9 KiB"
